Проверка подключения

После загрузки на Cloudflare нажмите проверку. Если переменные окружения заполнены, будет статус OK.

Автоматически отправляются: подтверждение записи, напоминание за день и за 2 часа. Вручную из карточки отправляются: перенос записи и благодарность с отзывом.

Важно: эта сборка содержит SMS-автоматизацию v67. В GitHub нужно загрузить содержимое архива в корень репозитория.

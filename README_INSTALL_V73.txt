СОЛНЦАНЕТ CRM V73 TARGET STYLE FULL

Загрузка на GitHub:
1. Распакуйте архив.
2. Откройте папку SOLNCANET_CRM_V73_TARGET_STYLE_FULL.
3. Загрузите на GitHub содержимое этой папки, не саму папку.
4. В корне репозитория должны лежать index.html, admin.html, zapis.html, assets, functions, docs, .github, service-worker.js, manifest.webmanifest.
5. Дождитесь деплоя Cloudflare Pages.
6. Откройте сайт в инкогнито или очистите кеш/service worker.

Эта сборка возвращает оформление главной страницы к версии f154e422.cloudflare-bx3.pages.dev и сохраняет последние функции CRM/SMS.

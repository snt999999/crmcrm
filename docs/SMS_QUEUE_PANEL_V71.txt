SMS очередь v71
===============

Автоматически в очередь попадают только:
1. Подтверждение записи
2. Напоминание за день
3. Напоминание за 2 часа

Только вручную отправляются из карточки заявки:
1. Перенос записи
2. Благодарность + отзыв

Во вкладке SMS очередь можно:
- смотреть все SMS по заявкам;
- менять время конкретной SMS;
- отменять конкретную SMS;
- возвращать отмененную SMS обратно;
- отправлять конкретную SMS сразу;
- запускать проверку очереди вручную.

Автоотправка работает через GitHub Actions:
.github/workflows/sms-reminders.yml

Без GitHub Actions или внешнего cron Cloudflare Pages Functions сами по времени не запускаются.

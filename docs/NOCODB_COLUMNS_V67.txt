========================================

Компания

Услуга
Адрес
м2

Комментарий администратора
Источник
DeletedAt
Дата удаления
Причина удаления

Для SMS отдельные колонки не нужны.
Служебные отметки отправки хранятся внутри поля "Комментарий администратора" в маркере:
[SOLNCANET_SMS_LOG:{...}]

Админка скрывает этот маркер от пользователя и из выгрузки.

Важно для Авто:
- Направление = Авто
- Авто = марка/модель автомобиля
- Авто услуги = JSON со строками услуг: name, material, price
- Общая стоимость = сумма по строкам услуг
- Адрес, м2, Итоговый м², Плёнка/Пленка/Материал очищаются, потому что для авто они не нужны.

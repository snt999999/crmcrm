Как полностью очистить GitHub и загрузить v67
=============================================

Вариант 1 — удалить содержимое, не удаляя репозиторий:
1. Откройте репозиторий GitHub.
2. Зайдите в папку, например assets.
3. Справа сверху нажмите кнопку с тремя точками "...".
4. Нажмите Delete directory.
5. Подтвердите Commit changes.
6. Так же удалите functions и другие папки.
7. Оставшиеся отдельные файлы можно открыть по одному и удалить через меню "..." / Delete file.
8. После очистки нажмите Add file -> Upload files и загрузите содержимое папки SOLNCANET_CRM_V67_SMS_AUTO_FULL.

Вариант 2 — самый чистый:
1. Repository -> Settings.
2. Внизу General найдите Danger Zone.
3. Delete this repository.
4. Создайте новый репозиторий с тем же именем.
5. Загрузите содержимое папки SOLNCANET_CRM_V67_SMS_AUTO_FULL.

Важно:
Загружайте не zip и не папку целиком, а именно файлы и папки изнутри:
index.html, admin.html, zapis.html, assets, functions, docs, .github и т.д.

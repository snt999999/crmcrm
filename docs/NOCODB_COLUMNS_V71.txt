Рекомендуемые колонки NocoDB для полной работы v71
==================================================

Основные:
- Id
- Дата записи
- Время записи
- Статус
- Направление
- Имя клиента
- Телефон
- Компания
- Ответственный
- Монтажники
- Услуга
- Адрес
- м2
- Итоговый м²
- Итоговый м2
- Материал
- Плёнка
- Пленка
- Комментарий клиента
- Комментарий администратора
- История изменений
- Источник
- Файлы

Для Авто:
- Авто
- Авто услуги
- Общая стоимость

Для Google Calendar, если будете возвращать старую синхронизацию:
- Google Calendar Event ID
- Ссылка на событие
- Cal Booking ID

Для SMS отдельная таблица не обязательна.
Служебные отметки об SMS хранятся скрытым маркером в поле Комментарий администратора:
[SOLNCANET_SMS_LOG:{...}]
В интерфейсе этот служебный текст скрывается.
